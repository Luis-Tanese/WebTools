RegEx Cheatsheet

. Any character except newline
\w Word character [a-zA-Z0-9_]
\d Digit [0-9]
\s Whitespace character
[abc] Any of a, b, or c
[^abc] Not a, b, or c
^ Start of string
$ End of string
* 0 or more
+ 1 or more
? 0 or 1
{n} Exactly n times
{n,} n or more times
{n,m} Between n and m times
(abc) Capture group
| Alternation (OR)
